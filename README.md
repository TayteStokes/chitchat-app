# ChitChat App

This is a small messaging platform application that was designed and developed by myself.

## Technologies

### Client

- React
- React Router
- Redux
- Styled Components

### Server

- Node
- Express
- Body-Parser
- Cookie-Parser
- Cors
- Massive
- Bcrypt
- AWS-SDK (S3)
- Passport
- Passport-Local
- Socket.io

### Database
- Cloud Based On Heroku
- PostgreSQL
